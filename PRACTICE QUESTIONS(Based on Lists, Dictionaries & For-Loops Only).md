# PYTHON PRACTICE QUESTIONS

### (Based on Lists, Dictionaries & For-Loops Only)

### Beginner → Intermediate Difficulty

## LISTS PRACTICE (20 Questions)

1.  Find the sum of all elements in a list.
2.  Find the maximum and minimum number in a list.
3.  Count how many times a specific number occurs in a list.
4.  Remove all occurrences of a given element from a list.
5.  Reverse a list without using reverse().
6.  Print only even numbers from a list.
7.  Print only odd numbers from a list.
8.  Given a list, create a new list with squares of each number.
9.  Find all numbers greater than a given value k.
10. Check if a list is sorted or not.
11. Find the second largest number in a list.
12. Count positive, negative, zero values in a list.
13. Merge two lists without using +.
14. Find the index of a given element (without using .index()).
15. Remove duplicates from a list.
16. Find common elements between two lists.
17. Move all zeroes to the end of the list.
18. Check if two lists are equal or not.
19. Find the frequency of each number in a list.
20. Find the running sum of a list. (Example: \[1,2,3\] → \[1,3,6\])

## DICTIONARIES PRACTICE (15 Questions)

1.  Count the frequency of each character in a string and store in a
    dictionary.
2.  Create a dictionary of numbers and their squares.
3.  Given a dictionary, print key with the highest value.
4.  Check if a key exists in a dictionary.
5.  Add a new key-value pair to a dictionary.
6.  Remove a key from a dictionary.
7.  Find the sum of all values in a dictionary.
8.  Swap keys and values in a dictionary.
9.  Merge two dictionaries.
10. Count how many values are even and odd in a dictionary.
11. Convert two lists into a dictionary.
12. Filter a dictionary: keep only items where value \> k.
13. Create a dictionary from a sentence with word frequencies.
14. Find the longest key and longest value (in length).
15. Create a nested dictionary and access specific values.

## FOR LOOP PRACTICE (15 Questions)

1.  Print numbers from 1 to 50 using a loop.
2.  Print the multiplication table of any number.
3.  Find the sum of digits of a number.
4.  Count vowels in a string.
5.  Print a pattern:
- *
- **
- ***
- ****
- *****


6.  Print reverse pattern: \*\*

-   *****
-   ****
-   ***
-   **
-   *   

7.  Print only characters at even index in a string.
8.  Print factorial of a number.
9.  Print all prime numbers between 1 and 50.
10. Count uppercase & lowercase letters in a string.
11. Calculate the product of all list elements.
12. Print elements of a list in reverse using loop (no reverse()).
13. Print Fibonacci series up to n terms.
14. Check if a number is palindrome using loop.
15. Print all substrings of a string.
